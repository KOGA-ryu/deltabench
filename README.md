# DeltaBench

DeltaBench turns code changes into structured risk evidence.

It answers:
- what changed
- why it matters
- what to review, test, or benchmark next

Primary commands:
```bash
deltabench diff --base main --head feature-x
deltabench review-packet --base main --head feature-x
```
